# Inbox

Drop unsorted research files here when migrating from other projects.

**Before the next render**, triage each file:

1. Determine the appropriate topic: `regulation/`, `deregulation/`, `markets/`, or `news/`
2. Move the file into that directory
3. Add a YAML frontmatter block (`title`, `date`) if missing
4. Register the file as a chapter in `_quarto.yml` under the correct part

Files left in `_inbox/` are excluded from the rendered book.
